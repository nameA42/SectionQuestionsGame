⚠ Firebase not configured yet — tap here for setup steps
  1. Go to console.firebase.google.com and create a project.
  2. Add a Web app and copy the config object.
  3. In the console go to Build → Realtime Database → Create database (Test mode is fine for classroom use).
  4. Open index.html, paste your config values where it says REPLACE_ME, and set FIREBASE_CONFIGURED = true.
  5. Push to GitHub — done!

Recommended Database Rules (Realtime Database → Rules tab):
{ "rules": { ".read": true, ".write": true } }
